injectParams is a helper function that enables the injection of route parameters as a signal. This approach is beneficial in a modern Angular signals-based architecture, as it allows for reactive handling of route parameters.

Usage

  • When called without any parameter, injectParams returns a signal containing the current route parameters.
  • To retrieve the value of a specific parameter, pass the parameter name to injectParams.
  • To transform the parameters, provide a function to injectParams.
